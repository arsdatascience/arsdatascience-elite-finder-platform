# Elite Finder App v1

## Visão Geral
O Elite Finder é uma plataforma abrangente para gestão de marketing digital, automação de vendas e criação de conteúdo.

## Módulos Principais

### 1. Dashboard & Analytics
Visão unificada de métricas de campanhas e desempenho.

### 2. Automação de Marketing
Ferramentas para criação e gestão de campanhas em múltiplas plataformas.

### 3. Geração de Imagens com IA (Novo!)
Um estúdio completo para criação de ativos visuais utilizando modelos de IA de última geração.
- **[Guia Completo de Geração de Imagens](./documentation/IMAGE_GENERATION.md)**
- Geração Text-to-Image (Flux, SDXL)
- Edição, Upscale e Variações
- Analytics de uso e créditos

### 4. Gestão de Chatbots
Configuração e treinamento de assistentes virtuais inteligentes.

## Instalação e Execução

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
